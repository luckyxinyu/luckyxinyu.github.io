1、状态量的传播

1.1 IMU测量模型

加速度的真值 = 加速度的测量值 + 加速度的零偏 + 加速度的噪声 + 旋转矩阵(I_G)*G坐标系下的重力加速度

角速度的真值 = 角速度的测量值 + 角速度的零偏 + 角速度的噪声

加速度零偏的导数 = 高斯分布(加速度)

角速度零偏的导数 = 高斯分布(角速度)

1.2 IMU连续运动模型

旋转矩阵的导数 = 旋转矩阵*角速度的反对称矩阵

位置的导数 = 速度

速度的导数 = 加速度
